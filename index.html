Recently completed work

In April 2015 I redesign “ www.springermedizin.de“.
Java, Spring FW, JSP, MySQL, CSS, HTML, Javascript and jQuery, XSLT. All the modules on the startpage are modular and can be reused with the CMS and are therefore reusable throughout the whole Springermedizin webpage.

In order to realise those tickets, I groomed the Project Manager.

Let me go a little into detail:

1) Hero Area “Aktuelles”

At the top you can see an output of a recent teaser. It is filled dynamically over the FAST Collection (Search Engine)

Style and formatting is done within the HTML, JSP and CSS. The output of grey trailerbox is done in the JSP and JS (jQuery).

2) The content for the collections “Medizin aktuell” as well as “Praxis and Beruf” are also is filled with FAST. FAST Parameters in the ECMS make sure that the number of hits and the filters (e. g. date) are controllable from the editor himself. FAST Implementation is done in Java. The orange “eMed” lock is in this case a check if the user has access to the article or not (Spring FW – Spring Acegi).

3) The “Banner” can be inserted over the CMS as well with a corresponding link. There is also some advertisement that is inserted via Google Doubleclick.

4) “CME” is short for Continuing Medical Education. German physicians have to gain points/credits over a certain period of time.

According to the profile (profession of the doctor), I output certain courses where he can gain his points. Here also all the formatting and styling is done via JSP, Javascript, jQuery, Ajax and CSS. Implementation in plain Java.

5) “Monster” Jobs are send via RSS-XML and then transformed with some Java and XSLT over the JSP into our webpage. I also provided the possibility for the editor to limit the number of entries/items, filter for certain physicians and set their own XSLT formatter for the output.

Implementation is done in Java (e. g. XSL Transformation). Used Java TLD’s to get the parameters for the JSPs. Formatting and styling is done via JSP, Javascript, jQuery and CSS.

6) Search Box

Search Box with directs to our search page with the according parameters. Whole styling of the search page is done by me as well.
